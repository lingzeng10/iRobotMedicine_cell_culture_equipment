✅ 好消息：您的手機可以連線
   這表示服務配置正確，問題應該是網路連線相關

❌ 朋友無法連線的可能原因
═══════════════════════════════════════════════════════════════

[1] 最常見：朋友的手機沒有連接到同一個 WiFi
───────────────────────────────────────────────────────────────
   請確認：
   ✓ 朋友的手機 WiFi 已開啟
   ✓ 朋友連接的 WiFi 名稱與您完全相同
   ✓ 不是不同的網路頻段（如 5G vs 2.4G，名稱相同但實為不同）

   注意：
   • 有些路由器會同時提供 "WiFi名稱" 和 "WiFi名稱-5G"
   • 兩者被視為不同的網路，無法互相訪問
   • 必須連接到完全相同的名稱

═══════════════════════════════════════════════════════════════

[2] 連接到訪客網路（Guest Network）
───────────────────────────────────────────────────────────────
   如果路由器有訪客網路功能：
   • 訪客網路通常被隔離，無法訪問主網路設備
   • 朋友的設備如果連到 "WiFi名稱-Guest"，無法訪問您的電腦

   解決方法：
   • 讓朋友連接到主 WiFi（不是訪客網路）

═══════════════════════════════════════════════════════════════

[3] AP 隔離（用戶端隔離）功能
───────────────────────────────────────────────────────────────
   某些路由器啟用了「AP 隔離」功能：
   • 會阻止同一個 WiFi 下的設備互相訪問
   • 即使在同一 WiFi 也無法透過區域網路連線

   解決方法：
   1. 登入路由器管理介面（通常是 http://192.168.0.1）
   2. 找到「無線設定」或「進階設定」
   3. 關閉「AP 隔離」或「用戶端隔離」功能

═══════════════════════════════════════════════════════════════

[4] 使用手機數據網路
───────────────────────────────────────────────────────────────
   如果朋友關閉 WiFi，使用 4G/5G 網路：
   • 無法訪問區域網路 IP (192.168.x.x)
   • 區域網路 IP 只能在同一個路由器下使用

   解決方法：
   • 確保朋友的手機 WiFi 已開啟並連接

═══════════════════════════════════════════════════════════════

📋 快速檢查步驟
═══════════════════════════════════════════════════════════════

步驟 1：請朋友檢查 WiFi 狀態
───────────────────────────────────────────────────────────────
   在朋友的手機上：
   1. 打開「設定」→「WiFi」
   2. 確認 WiFi 已開啟（不是使用數據網路）
   3. 查看目前連接的 WiFi 名稱
   4. 告訴您 WiFi 名稱
   5. 比對是否與您電腦連接的 WiFi 名稱完全相同

═══════════════════════════════════════════════════════════════

步驟 2：測試網路連線
───────────────────────────────────────────────────────────────
   在朋友的手機瀏覽器輸入：
   http://192.168.0.1
   
   如果可以看到路由器登入頁面：
   ✅ 表示在同一個網路
   
   如果無法訪問：
   ❌ 可能不在同一網路或有路由設定問題

═══════════════════════════════════════════════════════════════

✅ 解決方案
═══════════════════════════════════════════════════════════════

方案 A：讓朋友連接相同的 WiFi（最簡單）
───────────────────────────────────────────────────────────────
   1. 確認朋友知道正確的 WiFi 名稱和密碼
   2. 在朋友的手機上：
      • 開啟 WiFi 設定
      • 找到與您電腦相同的 WiFi 名稱
      • 輸入密碼連接
   3. 確保連接的是主 WiFi（不是 "名稱-Guest"）
   4. 等待連接成功後，再嘗試訪問網頁

═══════════════════════════════════════════════════════════════

方案 B：檢查路由器設定
───────────────────────────────────────────────────────────────
   如果確定在同一 WiFi 但仍無法訪問：
   
   1. 登入路由器管理介面
      地址通常是：http://192.168.0.1 或 http://192.168.1.1
      
   2. 檢查以下設定：
      • 「AP 隔離」或「用戶端隔離」→ 關閉此功能
      • 「訪客網路隔離」→ 確認設定
      • 「防火牆規則」→ 確認允許內網訪問
      
   3. 儲存設定並重新啟動路由器（如果需要）

═══════════════════════════════════════════════════════════════

💡 提示
═══════════════════════════════════════════════════════════════

   • 最常見的原因就是 WiFi 名稱不同或連接到訪客網路
   • 先確認朋友的手機連接的 WiFi 名稱
   • 如果名稱完全相同但還是無法訪問，可能是路由器設定問題

🔍 進階診斷
═══════════════════════════════════════════════════════════════

   如果朋友的手機支援，可以安裝網路工具 App：
   • Fing（免費網路掃描工具）
   • Network Analyzer
   
   這些工具可以：
   • 掃描區域網路中的設備
   • 查看是否能找到您的電腦 IP (192.168.0.186)
   • 確認是否在同一網路中
