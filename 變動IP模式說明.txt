═══════════════════════════════════════════════════════════════

✅ 變動 IP 模式的優點
═══════════════════════════════════════════════════════════════

1. 自動適應網路變更
   • 重新連接 WiFi 後，IP 可能改變
   • 系統會自動使用新的 IP
   • 無需手動修改任何設定

2. 無需維護配置文件
   • 不需要建立或維護 ip-config.txt
   • 適合在不同網路環境使用（公司、家裡等）

3. 簡單方便
   • 直接執行腳本即可
   • 自動偵測並顯示當前 IP

💡 使用範例
═══════════════════════════════════════════════════════════════

範例 1：正常使用
───────────────────────────────────────────────────────────────
   1. 雙擊執行：start-for-sharing.bat
   2. 系統自動偵測 IP，例如：192.168.0.186
   3. 顯示訪問地址：http://192.168.0.186:3000
   4. 分享給朋友即可

範例 2：IP 變動後
───────────────────────────────────────────────────────────────
   1. 重新連接 WiFi，IP 可能變成：192.168.0.200
   2. 重新執行：start-for-sharing.bat
   3. 系統自動偵測新 IP：192.168.0.200
   4. 自動使用新 IP，無需任何修改

🔄 如果未來想改成固定 IP
═══════════════════════════════════════════════════════════════

如果之後想要使用固定 IP：

步驟：
───────────────────────────────────────────────────────────────
   1. 在專案根目錄建立 ip-config.txt
   2. 在第一行輸入固定 IP，例如：192.168.0.186
   3. 使用 start-with-custom-ip.bat 啟動
   4. 系統會讀取配置文件中的固定 IP

═══════════════════════════════════════════════════════════════

⚠️ 注意事項
═══════════════════════════════════════════════════════════════

• 系統會自動偵測當前網路介面的主要 IP
• 如果有多個網路介面，會取得第一個找到的 IPv4 地址
• 每次執行都會重新偵測，確保使用最新的 IP
