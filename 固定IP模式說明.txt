🚀 如何使用固定 IP 模式
═══════════════════════════════════════════════════════════════

方式一：推薦使用
───────────────────────────────────────────────────────────────
   執行：start-with-custom-ip.bat
   
   特點：
   • ✅ 會讀取 ip-config.txt 中的固定 IP
   • ✅ 使用設定的固定 IP 地址
   • ✅ 不會自動變動
   
   使用方式：
   1. 確認 ip-config.txt 檔案存在
   2. 確認檔案內容為您要使用的 IP 地址
   3. 執行 start-with-custom-ip.bat
   4. 系統會使用配置文件中的固定 IP

═══════════════════════════════════════════════════════════════

方式二：一般啟動
───────────────────────────────────────────────────────────────
   執行：start-server.bat
   
   特點：
   • ✅ 會讀取 ip-config.txt 中的固定 IP（如果存在）
   • ✅ 如果沒有配置文件，則自動偵測
   • ✅ 目前有配置文件，所以使用固定 IP

═══════════════════════════════════════════════════════════════

✅ 固定 IP 模式的優點
═══════════════════════════════════════════════════════════════

1. IP 地址穩定
   • 每次啟動都使用相同的 IP
   • 分享給朋友後，網址不會改變
   • 適合需要穩定網址的場合

2. 可預測性高
   • 知道確切的訪問地址
   • 可以提前分享網址給朋友
   • 不需要每次都重新檢查 IP

3. 適合固定環境
   • 在公司或固定的網路環境使用
   • IP 地址很少變動的情況

📝 如何修改固定 IP
═══════════════════════════════════════════════════════════════

如果您的 IP 地址改變了，或想使用不同的 IP：

步驟 1：編輯配置文件
───────────────────────────────────────────────────────────────
   1. 打開 ip-config.txt 檔案
   2. 修改第一行的 IP 地址
   3. 例如改為：192.168.1.100
   4. 儲存檔案

步驟 2：重新啟動服務
───────────────────────────────────────────────────────────────
   1. 停止當前運行的服務
   2. 重新執行 start-with-custom-ip.bat
   3. 系統會使用新的 IP 地址

═══════════════════════════════════════════════════════════════

📄 配置文件格式
═══════════════════════════════════════════════════════════════

ip-config.txt 檔案格式：
───────────────────────────────────────────────────────────────
   192.168.0.186
   
   說明：
   • 第一行為要使用的固定 IP 地址
   • 只包含 IP 地址，不含協議或端口
   • 格式：xxx.xxx.xxx.xxx
   
   有效範例：
   192.168.0.186
   192.168.1.100
   10.0.0.5
   
   無效範例：
   http://192.168.0.186    ❌（不要包含協議）
   192.168.0.186:3000      ❌（不要包含端口）

💡 使用建議
═══════════════════════════════════════════════════════════════

• 固定環境：使用固定 IP 模式（當前設定）

⚠️ 注意事項
═══════════════════════════════════════════════════════════════

• 如果您的 IP 地址實際改變了（例如重新連接 WiFi），

• 建議：
  - 如果 IP 經常變動，可以考慮使用變動 IP 模式
  - 如果 IP 穩定，固定 IP 模式更方便
