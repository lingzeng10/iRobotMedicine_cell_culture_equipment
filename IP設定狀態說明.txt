✅ 當前狀態：已設定為變動 IP（自動偵測模式）
═══════════════════════════════════════════════════════════════

系統目前配置為自動偵測 IP 地址模式，無需手動設定！

═══════════════════════════════════════════════════════════════

📋 可用的啟動方式
═══════════════════════════════════════════════════════════════

[方式 1] 推薦：start-for-sharing.bat
───────────────────────────────────────────────────────────────
   特點：
   • ✅ 完全自動偵測 IP
   • ✅ 每次執行都會取得最新 IP
   • ✅ 最適合變動 IP 環境
   
   使用方式：
   1. 雙擊執行 start-for-sharing.bat
   2. 系統自動偵測並顯示當前 IP
   3. 無需任何額外設定

═══════════════════════════════════════════════════════════════

[方式 2] start-server.bat
───────────────────────────────────────────────────────────────
   特點：
   • ✅ 自動偵測 IP（如果沒有配置文件）
   • ✅ 如果存在 ip-config.txt，會優先使用固定 IP
   • ✅ 目前沒有配置文件，所以使用自動偵測
   
   使用方式：
   1. 雙擊執行 start-server.bat
   2. 系統會自動偵測 IP
   3. 如果未來建立 ip-config.txt，則使用固定 IP

═══════════════════════════════════════════════════════════════

[方式 3] start-with-custom-ip.bat
───────────────────────────────────────────────────────────────
   特點：
   • ✅ 優先讀取配置文件（如果存在）
   • ✅ 沒有配置文件時自動偵測
   • ✅ 目前沒有配置文件，所以使用自動偵測
   
   使用方式：
   1. 目前沒有配置文件，會自動偵測
   2. 如果建立 ip-config.txt，則使用固定 IP

═══════════════════════════════════════════════════════════════

🔄 如何確認使用變動 IP？
═══════════════════════════════════════════════════════════════

檢查方式：
───────────────────────────────────────────────────────────────
   1. 確認專案根目錄沒有 ip-config.txt 檔案
   2. 執行任何啟動腳本
   3. 查看輸出訊息，應該顯示「自動偵測到 IP」或類似訊息

═══════════════════════════════════════════════════════════════

  → 系統會使用自動偵測模式

• start-for-sharing.bat ✅
  → 完全自動偵測，不受配置文件影響

• start-server.bat ✅
  → 會檢查配置文件，目前使用自動偵測

• start-with-custom-ip.bat ✅
  → 會檢查配置文件，目前使用自動偵測

💡 變動 IP 模式的好處
═══════════════════════════════════════════════════════════════

✅ 自動適應網路變更
   • 重新連接 WiFi 後，IP 可能改變
   • 系統會自動使用新的 IP

✅ 無需手動維護
   • 不需要每次修改配置文件
   • 適合在不同網路環境使用

✅ 簡單方便
   • 直接執行腳本即可
   • 無需額外設定

📝 如果未來想改成固定 IP
═══════════════════════════════════════════════════════════════

如果您之後想要使用固定 IP：

步驟 1：建立配置文件
───────────────────────────────────────────────────────────────
   在專案根目錄建立 ip-config.txt
   內容範例：192.168.0.186

步驟 2：選擇啟動方式
───────────────────────────────────────────────────────────────
   • 使用 start-with-custom-ip.bat（推薦）
   • 或使用 start-server.bat（也會讀取配置文件）
